Projects

These are some of the hobby projects I've been working on.

Gitlab-Team-Dashboard

A dashboard that displays the status of development projects managed in GitLab (on-premise or in the cloud). The dashboard is designed to be displayed on a TV screen or some large display.

Retro-Carnage

A work-in-progress project building a modern multidirectional scrolling shooter. Once finished, Retro-Carnage is going to be a worthy successor of classic video games like Ikari Warriors, War Zone, or Dogs of War.

MailBackup

Versatile commandline tool to download emails from an IMAP server and store them as EML files on your computer.

Salesforce-Backup

This tool performs automated backups of SalesForce.com export data and stores it locally or to either AWS or Azure.

Breakout-Go

Breakout-GO is a Breakout clone for the ODROID-GO Game Kit written in C++.

Hex editor Plugin for NetBeans

This project provided a hex editor for the NetBeans IDE with text preview in different encodings.

MongoDB Plugin for NetBeans IDE

This project provided a MongoDB integration in NetBeans when better tools where missing.

BeanShell Plugin for NetBeans IDE

This project provided an interactive shell (based on BeanShell) in NetBeans before JShell existed.

Grab2Web

This was one of my hobby projects back in 2004. The tools captures images from a webcam and uploads them to a FTP server.

Klickus

A Clickomania clone written in Delphi
